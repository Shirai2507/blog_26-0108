## レイアウト作成&継承

【@extends】<br>
このテンプレート(子)では別ファイル(親)のレイアウトを継承してレンダリングする。
- 継承元(親): resources\views\layouts\sample.blade.php
```
{{-- 継承先(子) | resources\views\outputs\example.blade.php --}}
@extends('layouts.sample')
```
<br>

【@section, @yield】<br>
レイアウト継承元(親)の「@yield('title')」部分に、継承先(子)で指定した文字列「このページのタイトル」を当てはめる。
```
{{-- (親) | resources\views\layouts\sample.blade.php --}}
<head>
    <title>@yield('title')</title>
```
```
{{-- (子) | resources\views\outputs\example.blade.php --}}
@extends('layouts.sample')
@section('title', 'このページのタイトル')
```
<br>

【@section, @endsection, @yield】<br>
レイアウト継承元(親)の「@yield('news')」部分に、継承先(子)の「@section('news')」～「@endsection」の中身を当てはめる。
```
{{-- (親) | resources\views\layouts\sample.blade.php --}}
<div class="news">@yield('news')</div>
```
```
{{-- (子) | resources\views\outputs\example.blade.php --}}
@extends('layouts.sample')
@section('news')
    <p>2026/1/15 宝くじが当たった</p>
    <p>2026/1/13 保有株が暴騰した</p>
@endsection
```
<br>

【@show, @parent】<br>
レイアウト継承元(親)の「@section('lucky-charm')」～「@show」の中身を、継承元(子)の「@parent」部分に当てはめる。<br>
つまり出力すると「@section('lucky-charm')」～「@show」の部分が、「一富士二鷹三茄子」のように表示される。
```
{{-- (親) | resources\views\layouts\sample.blade.php --}}
@section('lucky-charm')
    <span>一富士</span>
    <span>二鷹</span>
@show
```
```
{{-- (子) | resources\views\outputs\example.blade.php --}}
@extends('layouts.sample')
@section('lucky-charm')
    @parent
    <span>三茄子</span>
@endsection
```

## コンポーネント

【@component】<br>
別ファイル「sample.blade.php」の全コード(コンポーネント)(部品)を「@component('components.sample')」～「@endcomponent」部分(差込口)に当てはめる。
```
{{-- 部品(呼ばれる方) | resources\views\components\sample.blade.php (全コード) --}}
<span class="message">これがメッセージ</span>
```
```
{{-- 部品の差込口(呼ぶ方) | resources\views\outputs\example.blade.php --}}
@extends('layouts.sample')
@component('components.sample')
@endcomponent
```
<br>

【{{ $slot }}】<br>
「@component('components.sample')」～「@endcomponent」の中身の「これがメッセージ。」を「{{ $slot }}」に代入したうえで部品を当てはめる。<br>
つまり「```<span class="message">これがメッセージ</span>```」が差し込まれる。
```
{{-- 部品(呼ばれる方) | resources\views\components\sample.blade.php (全コード) --}}
<span class="message">{{ $slot }}</span>
```
```
{{-- 部品の差込口(呼ぶ方) | resources\views\outputs\example.blade.php --}}
@extends('layouts.sample')
@component('components.sample')
    これがメッセージ。
@endcomponent
```
<br>

【@slot】<br>
「@slot('message1')」～「@endslot」中身の「1行目のメッセージ。」も「{{ $message1 }}」に代入したうえで部品を当てはめる。<br>
つまり「```<span class="message1">1行目のメッセージ。</span><span class="message2">2行目のメッセージ。</span>```」が差し込まれる。
```
{{-- 部品(呼ばれる方) | resources\views\components\sample.blade.php (全コード) --}}
<span class="message1">{{ $message1 }}</span>
<span class="message2">{{ $slot }}</span>
```
```
{{-- 部品の差込口(呼ぶ方) | resources\views\outputs\example.blade.php --}}
@extends('layouts.sample')
@component('components.sample')
    @slot('message1')
        1行目のメッセージ。
    @endslot
    2行目のメッセージ。
@endcomponent
```
<br><br>

## サブビュー

【@include】<br>
別ファイル「sample.blade.php」の全コード(サブビュー)を「@include()」部分に当てはめる。<br>
これは「{{ $message1 }}」に「1行目のメッセージ。」を代入したうえで行われる。<br>
※コンポーネントのスロットのような機能はない。
```
{{-- 呼ばれる方(サブビュー) | resources\views\sub-views\sample.blade.php (全コード) --}}
<span class="message1">{{ $message1 }}</span>
<span class="message2">2行目のメッセージ。</span>
```
```
{{-- 呼ぶ方 | resources\views\outputs\example.blade.php --}}
@extends('layouts.sample')
@include('sub-views.sample', ['message1' => '1行目のメッセージ。'])
```
<br><br>

## コレクションビュー

【@each】<br>
別ファイル(ビュー)の全コードを繰り返し埋め込む。<br>
- 第1引数: 繰り返し埋め込むビュー
- 第2引数: 埋め込むビューの変数に代入する値を用意する配列。この配列の要素数が繰り返し回数とイコール
- 第3引数: 埋め込むビューの変数と配列の要素を紐づける。各要素が繰り返しこの名前の変数(ここではcomment)代入される。
- 第4引数: 配列が空の場合に読み込むビューの指定。
```
{{-- 繰り返し呼ばれるビュー | resources\views\each-items\sample.blade.php (全コード) --}}
<div class="comment">
  <p class="author">{{ $comment->user_name }}</p>
  <p class="body">{{ $comment->body }}</p>
</div>
```
```
{{-- 配列が空の場合に呼ばれるビュー | resources\views\each-items\empty.blade.php (全コード) --}}
<p class="empty">コメントはまだありません。</p>
```
```
{{-- 呼ぶ側 | resources\views\outputs\example.blade.php --}}
@extends('layouts.sample')
@php
    $comments = [
        ['user_name' => 'タロウ', 'body' => 'こんにちは！'],
        ['user_name' => 'ハナコ', 'body' => 'いい記事でした。'],
        ['user_name' => 'ジロウ', 'body' => '質問いいですか？'],
    ];
@endphp
@each('each-items.sample', $comments, 'comment', 'partials.comment_empty')
```
>結果、「@each()」部分に嵌め込まれるコード
>```<h1>コメント一覧</h1>
><div class="comment">
>  <p class="author">タロウ</p>
>  <p class="body">こんにちは！</p>
></div>
><div class="comment">
>  <p class="author">ハナコ</p>
>  <p class="body">いい記事でした。</p>
></div>
><div class="comment">
>  <p class="author">ジロウ</p>
>  <p class="body">質問いいですか？</p>
></div>```
